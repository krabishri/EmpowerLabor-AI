🤖 Job Mentor Agent

AI-powered career guide for India's informal workers — built with Python Flask + IBM Watsonx.ai Granite Models

💼 Job Matching 🎓 Upskilling 🛡️ Welfare Schemes 🎙️ Voice Input ♿ Accessible 🌐 11 Languages

Features

🤖

Mentor Chat

Warm, multilingual career guidance. Responds in the same language as the user — Hindi, Tamil, Telugu, and 8 more.

💼

Job Matching

Localized daily-wage & gig job suggestions with daily ₹ wage, required skills, physical demand, and seasonal availability.

🎓

Upskilling Roadmap

3-step learning paths mapped to IBM SkillsBuild, PMKVY, NSDC, and state skill missions. Tracks completion with checkboxes.

🛡️

Welfare Eligibility

Plain-language breakdown of PM-KISAN, MGNREGS, Ayushman Bharat, PM SVANidhi, Ujjwala Yojana, and more.

🎙️

Voice Profile

Large tap-friendly mic buttons for voice input. Supports all 11 Indian languages via Web Speech API.

♿

Full Accessibility

High-contrast mode, large-text toggle, ARIA live regions, screen reader announcements, skip links, focus management.

Project Structure

  • job_mentor_agent/
    • app.py ← Flask backend + AGENT_INSTRUCTIONS
    • requirements.txt ← 5 Python packages
    • .env.example ← Credential template (copy to .env)
    • README.md ← Full deployment guide
    • templates/
      • index.html ← Chat interface
      • dashboard.html ← Job / Upskill / Welfare dashboard
    • static/css/
      • style.css ← Bootstrap 5 + custom responsive styles
    • static/js/
      • app.js ← Chat, voice input, mode switching, profile
      • dashboard.js ← Dashboard panels, course tracker, voice

🔧 AGENT_INSTRUCTIONS — Customization Block (top of app.py)

  • TONE & PERSONA — Speaking style, reading level, empathy cues
  • MULTILINGUAL PREFERENCES — Supported languages, auto-detect, code-switching rules
  • DOMAIN FOCUS — Priority sectors, job parameter format, regional focus
  • UPSKILLING — Course platforms (IBM SkillsBuild, PMKVY, NSDC), format rules
  • WELFARE SCHEMES — Which schemes to cover, output format, eligibility disclaimer
  • ENTERPRISE SAFETY RULES — Content blocks, distress routing, PII handling

Quick Start (5 Steps)

1

Create virtual environment

python -m venv venv && venv\Scripts\activate
2

Install dependencies

pip install -r requirements.txt
3

Configure credentials

Copy .env.example → .env and fill in your IBM Cloud API Key and Watsonx Project ID.

4

Run the app

python app.py
5

Open in browser

Go to http://localhost:5000 — fill profile → choose mode → chat or speak!
